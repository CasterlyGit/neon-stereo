// Retro aesthetic

Scanlines via repeating linear gradient. Multi-layer text-shadow for the neon glow stack. CRT vignette via radial gradient. Flicker via 5s keyframe. Pure CSS — no shaders.

// Provider-agnostic

Spotify uses PKCE + the typed /v1/me/player client. YouTube runs in an IFrame Player API embed. Both funnel into the same PlaybackState stream.

// Security posture

Bearer tokens stay in main process — never enter the renderer. Refresh tokens stored in macOS Keychain via keytar, with a 0600 JSON fallback if the native binding fails.
